# Source Document Test Fixtures

Small, deterministic files for Source Document upload, preview, and extraction tests.

## Alpha fixtures

- `alpha-embedded-text-lesson.pdf` — one-page PDF with an embedded text layer.
- `alpha-lesson-notes.txt` — plain-text lesson notes.

## Full fixtures

- `full-lesson-document.docx` — DOCX lesson document with headings, bullets, a table, and questions.
- `full-lesson-slides.pptx` — PPTX slide deck with text content.
- `full-vocabulary.csv` — CSV vocabulary table for preview/extraction tests.

These files are intentionally tiny and contain synthetic lesson content only.
